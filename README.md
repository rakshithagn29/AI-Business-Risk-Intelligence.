# 🚀 AI Driven Business Risk Intelligence
## Customer Risk Assessment & Churn Prediction

### 📌 Problem Statement
Businesses lose crores of rupees every year because 
they discover customer risk and churn TOO LATE.

### 💡 Solution
An AI powered platform that predicts customer risk 
and churn BEFORE it happens using advanced ML models.

### ✨ Key Features
- Multi Dimensional 5D Risk Scoring Engine
- Triple Horizon Churn Prediction (30/60/90 days)
- Real Time Social Sentiment Fusion
- Explainable AI using SHAP and LIME
- What-If Action Simulator
- Revenue at Risk Calculator
- Industry Intelligence Adapter

### 🛠️ Tech Stack
- Python 3.14
- XGBoost, Random Forest, Scikit-learn
- SHAP, LIME
- FastAPI, Streamlit
- MongoDB, PostgreSQL
- Docker, AWS

### 🚀 How to Run
1. Clone this repository
2. Create virtual environment: python -m venv venv
3. Activate: venv\Scripts\activate
4. Install libraries: pip install -r requirements.txt
5. Run dashboard: streamlit run dashboard/app.py
